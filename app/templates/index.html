Warmane Auction House History

This is a simple website build using Python and SQLite that keeps track of prices on the auction house. Warmane hosts private World of Warcraft servers for multiple expansions. This website is in no way involved with the official Warmane. The data is imported from auctionator scans and the website is hosted with Flask, uWSGI and NGINX.

Development is still in progress. Source code will be made public on GitHub eventually.
