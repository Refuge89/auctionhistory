WoW Auction House History

Browse the price history of several private World of Warcraft auction houses from Warmane and Gamer District.

This website is a personal project created with Python and SQLite and it is hosted with Flask, uWSGI and NGINX. I am not affiliated with any of private servers featured on this website.

Development is still in progress. Source code will be made public on GitHub eventually.
